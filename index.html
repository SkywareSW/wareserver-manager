Features

Everything you need.
Nothing you don't.

Built for people who just want their server running without touching a terminal.

⚡
One-click launch
Start and stop your Paper or Spigot server instantly. No commands, no setup scripts, just a button.
📟
Live console
Stream your server logs in real time and send commands directly from the app. Full colour output.
📁
File manager
Browse your server folder, upload plugins via drag and drop, and delete files — all without leaving the app.
🌐
Built-in tunneling
Integrated playit.gg support. Let friends join without port forwarding — just install the agent and share your address.
🎛
RAM slider
Drag to allocate memory. Settings are saved between sessions so you never have to configure the same thing twice.
💾
Persistent config
Your server path, port, RAM, and gameplay settings are remembered automatically across restarts.
Getting started

Up and running
in minutes.

Download, configure, launch. That's it.

1
Download WareServer
Grab the latest installer from the releases page and run it.
→ Download .exe
2
Download a server JAR
Grab the latest Paper JAR from papermc.io and put it in a folder.
→ papermc.io/downloads
3
Configure & launch
Click Configure, browse to your JAR and server folder, then hit Launch.
4
Invite friends (optional)
Head to the Tunnel tab, install the playit agent, and share your address. No port forwarding needed.
